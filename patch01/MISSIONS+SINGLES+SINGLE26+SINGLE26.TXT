{A} An zwei Fronten 
 
Ihre Aufgabe ist es, die Allianz von Österreich, England und Dänemark zu vernichten. 
Sie sollen die Österreicher, die eine der östlichen Inseln besiedelt haben (auf der Karte gibt es insgesamt vier Inseln), besiegen.
Sie werden durch Truppen der nördlichen Front unterstützt. Dort kämpft Ihr Kamerad, ein weiterer spanischer Offizier. Sie sollen ihm Ressourcen bringen. 
Ihre Verbündeten, die Sachsen, können Ihnen auch Hilfe leisten. 
Sie verlieren, wenn Ihr Hauptcharakter ums Leben kommt oder wenn die Truppen an der nördlichen Front eine Niederlage erleiden. 
Nur der Hauptcharakter kann sich mit Verbündeten unterhalten. 
 
Gegenwärtiges Ziel:
{B}Besiegen Sie Österreich auf einer der östlichen Inseln. 
{C}~Besiegen Sie Österreich auf einer der östlichen Inseln.~
{D}Vernichten Sie die englischen Werften auf den östlichen Inseln (zwei auf der nördlichen Insel und zwei auf der südlichen), deren Schiffe Ihre Einrichtungen überfallen. 
{E}~Vernichten Sie die englischen Werften auf den östlichen Inseln (zwei auf der nördlichen Insel und zwei auf der südlichen), deren Schiffe Ihre Einrichtungen überfallen.~
{F}Zerstören Sie das Banditenlager, das nördlich von Ihnen liegt. 
{G}~ Zerstören Sie das Banditenlager, das nördlich von Ihnen liegt.~
{K} Zerstören Sie das Banditenlager auf der Insel, die nördlich von Ihnen liegt. 
{L}~Zerstören Sie das Banditenlager auf der Insel, die nördlich von Ihnen liegt.~
{M} Zusätzliches Ziel: 
{N} Bringen Sie dem Befehlshaber der nördlichen Front je 5000 aller Ressourcenarten (auf die Insel, die sich nördlich von Ihrer befindet). 
{O}~Bringen Sie dem Befehlshaber der nördlichen Front je 5000 aller Ressourcenarten (auf die Insel, die sich nördlich von Ihrer befindet).~

Besuchen Sie Ihren Verbündeten Sachsen im Fort nördlich von Ihnen, damit sie Ihnen helfen, die Angriffe der Engländer abzuwehren!
Damit die Engländer völlig mit Angriffen aufhören, müssen Sie ihre Werften auf den  Inseln vernichten.
