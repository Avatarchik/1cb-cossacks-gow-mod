#BRIEFING faces 0
{A} Weltkrieg
 
Ihre Aufgabe ist es, die Allianz von Dänemark, Sachsen und Portugal zu zerschlagen. Dafür muss Portugal auf der Insel nordöstlich von Ihnen vernichtet werden. 
Leider können Ihre Arbeiter keine Artilleriedepots bauen, deshalb müssen Sie die notwendigen Pläne beim Feind auf dem Festland finden. 
Sie erleiden eine Niederlage, wenn Sie das Rathaus mit dem Staatsschatz verlieren oder wenn Sie die Pläne nicht bekommen. 

Gegenwärtiges Ziel:
{B} Vernichten Sie Portugal auf der Insel nordöstlich von Ihnen. 
{C}~Vernichten Sie Portugal auf der Insel nordöstlich von Ihnen.~
{D} Finden Sie die Pläne für das Artilleriedepot in einem der Rathäuser von Dänemark, Sachsen oder Portugal (auf dem Festland).
{E}~Finden Sie die Pläne für das Artilleriedepot in einem der Rathäuser von Dänemark, Sachsen oder Portugal (auf dem Festland).~
