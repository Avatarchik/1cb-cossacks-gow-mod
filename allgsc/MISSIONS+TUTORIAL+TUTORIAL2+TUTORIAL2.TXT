#PAGE9 Missions\Pictures\russia 8
Sie können Ihren Truppen befehlen, beliebige Objekte Ihrer Siedlung zu schützen. Wählen Sie dazu 2 oder 3 Soldaten, zum Beispiel Pikeniere. Unten links im Interface sehen Sie das Icon für den Befehl "Bewachen". Klicken Sie darauf, damit sich der Mauszeiger in einen Schild verwandelt. Wenn Sie den Mauszeiger auf der Karte bewegen, sehen Sie, dass der Schild grau wird, wenn es in seiner gegenwärtigen Position nichts gibt, das man bewachen könnte. Sobald Sie den Mauszeiger auf ein Objekt richten, das bewacht werden kann, erscheint der Schild in Farbe. Wenn Sie nun den Soldaten mit einem Linksklick befehlen, das Objekt zu bewachen, werden sie dies tun, bis Sie ihnen einen anderen Befehl erteilen.

#PAGE14 Missions\Pictures\russia 20
Nun können Sie eine Schwadron erstellen. Dazu benötigen Sie 36 Pikeniere, einen Kommandanten und einen Trommler. Versammeln Sie all diese Einheiten und wählen Sie den Kommandanten. Unten links im Interface sehen Sie nun Icons für Formationen. Es gibt drei davon, von denen jede einer bestimmten Formation entspricht, nämlich der Linien-, Säulen- und Karreeformation. Wählen Sie die gewünschte Formation. Außer der Formation mit 36 Einheiten stehen Ihnen auch Formationen mit 15, 72, 120 und 196 Einheiten zur Verfügung.
#PAGE15 Missions\Pictures\russia 16
Wenn Sie sehen, dass ein zahlenmäßig überlegener Feind Sie angreift und Sie glauben, dass Sie die Schlacht verlieren könnten, können Sie Ihre Bauern zum Angriff einsetzen. Natürlich sind Bauern weniger effektiv als reguläre Truppen, ihr Einsatz könnte aber dennoch für den Ausgang der Schlacht entscheidend sein. Wenn Sie mit Bauern angreifen, sollten Sie sicherstellen, dass Ihre Truppen in der Nähe sind. Ansonsten könnten die Bauern gefangen genommen werden.
#PAGE24 Missions\Pictures\mord2_tut 2
